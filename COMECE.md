## 🎯 Para Começar Agora

### Terminal 1 - Abra o servidor backend
```bash
cd backend
npm run dev
```

**Esperado:** Você verá `✅ Servidor rodando em http://localhost:3001`

---

### Terminal 2 - Abra o servidor frontend
```bash
cd frontend
npm run dev
```

**Esperado:** Você verá `Local: http://localhost:3000`

## ⚠️ Se Tiver Erros

**Erro "Cannot find module express"**
- Execute: `npm install` na pasta do erro

**Porta 3000 ou 3001 em uso**
- Mude em `frontend/vite.config.js` ou `backend/src/server.js`

**Banco não criado**
- Isso é normal! Será criado automaticamente na primeira execução
